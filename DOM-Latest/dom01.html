addEventListener: used to attach an event handler to an HTML element. It allows your webpage to respond to user actions like clicks, typing, mouse movement, etc. Syntax: element.addEventListener(event, function, useCapture); DOMContentLoaded is an event that fires when: 👉 The HTML document is fully loaded and parsed 👉 Before images, CSS, and external resources finish loading Syntax: document.addEventListener("DOMContentLoaded", function() { // your code here }); querySelector is used to select the first HTML element that matches a CSS selector. Syntax: document.querySelector("CSS_selector");
